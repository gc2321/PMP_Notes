--------------------------------
7.Communication Management
--------------------------------

- use of formal written communication is most useful in complex problems
- status meeting: exchange information about the project

**** Plan Communication Management
- communication channels: n(n-1)/2

**** Manage Communication
- meetings, project reporting
- Project Reporting:
Status report, Trend report, Forecasting report, Variance report, Earned value report, Lessons learned

- performance reporting: (1) change requests (2) performance reports

**** Monitor Communication
- Exception report: completed when variances exceed a given limit
